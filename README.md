# Super Copy - Plataforma Web

¡Bienvenido al repositorio de **Super Copy**! Este proyecto consiste en el desarrollo de un sitio web integral para una papelería y centro de servicios de impresión, diseñado con un enfoque en la eficiencia, la claridad informativa y una identidad visual dinámica.

## Descripción del Proyecto
Super Copy resuelve la falta de información digital sobre productos y servicios escolares, de oficina y de regalo. La plataforma permite a los usuarios consultar catálogos, listas de precios de servicios técnicos y contactar directamente con el establecimiento.

### Objetivos
* **Digitalización:** Centralizar la oferta de productos y servicios.
* **Eficiencia:** Optimizar la comunicación con estudiantes y padres de familia.
* **Identidad:** Reforzar la marca mediante una paleta de colores vibrante (Azul, Amarillo y Negro).

---

## Tecnologías Utilizadas

* **HTML5:** Estructura semántica completa (`<header>`, `<main>`, `<section>`, `<aside>`, `<nav>`, `<footer>`).
* **CSS3:** Diseño responsivo, Flexbox, Grid, y uso de pseudoclases (`:hover`, `:focus`).
* **XML & DTD:** Almacenamiento de datos estructurados para el catálogo de productos.
* **Git & GitHub:** Control de versiones y despliegue.

---

## Estructura del Repositorio

* `/css`: Contiene los estilos externos (`estilos.css`).
* `/data`: Archivos de datos estructurados (`catalogo.xml` y `catalogo.dtd`).
* `/img`: Recursos visuales y logo de la marca.
* `index.html`: Página principal y lista de precios.
* `productos.html`: Catálogo con barra lateral de categorías.
* `contacto.html`: Formulario completo con validaciones HTML5.
